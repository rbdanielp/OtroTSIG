Instructivo para copiar la configuracion de capas, agrupaciones y estilos

1- Configurar datasource para que tome los datos geograficos desde la base de datos
2- Copiar el contenido de CapasPostgis.tar.gz en el directorio /data_dir/workspaces de geoserver


Levantar geoserver y validar que la configuracion sea la correcta:

Layer preview - buscar "CapasPostgis:CapaBaseYPOIS" y clic en OpenLayers.
